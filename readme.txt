== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/woo-genie-checking` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Enable this gateway from WooCommerce settings /wp-admin/admin.php?page=wc-settings&tab=checkout&section=geniechecking
